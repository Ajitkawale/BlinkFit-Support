# BlinkFit Accessibility Features

BlinkFit is designed to be usable by everyone. Here are the built‑in accessibility features:

## One‑Eye Mode
If you have limited vision in one eye or wear glasses that interfere with blink detection, enable **One‑Eye Mode** in Settings → Accessibility. The blink detector will then track only one eye.

## VoiceOver
All buttons and controls have descriptive labels. VoiceOver users can navigate the app with standard gestures.

## Haptic Feedback
When haptics are enabled in Settings, you’ll feel a gentle tap when a blink is detected, a fruit is cut, or a session completes. This provides confirmation without relying on sound.

## Colour Filters
Choose from red/green, blue/yellow, or grayscale filters in Settings → Accessibility to adjust the app’s colour scheme for your visual needs.

## Camera‑Free Alternatives
If you prefer not to use the camera or if detection is difficult, you can complete exercises using on‑screen timers. Tap “Skip” or use the manual timer option in the Palming exercise.
